8. Дано описание:
N DB ? ; число без знака
K DW ?
Вычислить: K ≔ N(N+1)/2
```

```
Ответ:
MOV AL, N
INC AL ; AL = N+1
MUL N ; AX = (N+1)N
MOV BL, 2
DIV BL ; AL = (N+1)N/2
CBW ; AX = (N+1)N/2
MOV K, AX
```

9. Пусть A, B, C и X — знаковые байтовые переменные, а Y — переменная типа DWORD. Вычислить: Y ≔ AX^2 + BX + C
```

```
Ответ:

A DB ?
B DB ?
C DB ?
X DB ?
Y DD ?

MOV AL, A
MUL X ; ax = A*X
MUL X ; ax = A*X*X
MOV BX, AX ; bx = ax = A*X*X
MOV AL, B ; al = B
MUL X ; ax = B*X
ADD BX, AX ; bx = bx + ax = A*X*X + B*X
ADD BX, C ; bx = A*X*X + B*X + C
MOV Y, BX ; Y = bx
```
